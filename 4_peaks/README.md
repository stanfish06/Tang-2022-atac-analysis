# Folder for peak calling
Scripts intro
-------------
| Name                 | Description                                    |
| -------------------- | ---------------------------------------------- |
| `find-atac-peaks.sh` | Call peaks using MACS2                         |
| `filter-peaks.sh`    | Filter peaks based on quality and width        |


Instruction
-----------
- First, run `find-atac-peaks.sh` to call peaks
- Next, run `filter-peaks.sh` to filter the raw peaks
